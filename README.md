# Obsidian Vault: The Roottrees are Dead

## What is this?
This is a collection of notes and files related to the video game, "The Roottrees are Dead".

## How to use this?
This is a Obsidian vault, so you can use the Obsidian app to view the notes.

## How to install Obsidian?
- Download the Obsidian app from the Obsidian website: https://obsidian.md/
- Install the app
- Open the app and select "Open Vault"
- Select the "The Roottrees are Dead" vault
